# **Cisco Packet Tracer Projects**

##  Overview

This repository contains a collection of **Cisco Packet Tracer projects** designed to practice and understand **core networking concepts** through hands-on labs.

The projects focus on **realistic network scenarios**, making this repository useful for learning, revision, and certification preparation.

---

##  Purpose

The main goals of this repository are to:

- Provide **practical networking examples**
- Improve understanding of **Cisco network configurations**
- Support **CCNA-level learning and practice**
- Offer reusable **Packet Tracer labs**

---

##  Repository Structure

Projects are organized into folders, each representing a specific networking topic.

### Covered topics include:

- **IP addressing & subnetting**
- **Static and dynamic routing**
- **Router and switch topologies**
- **VLAN configurations**
- **Network services (DHCP, DNS)**
- **Advanced network designs**

Each project usually includes:
- **`.pkt` files** compatible with Cisco Packet Tracer
- Preconfigured network devices
- Clear and readable setups

---

##  Projects

| **Project** | **Description** |
|------------|-----------------|
| **FULL_MESH** | Full mesh topology connecting multiple routers |
| **OSPF** | Dynamic routing configuration using **OSPF** |
| **SUBNETTING** | Examples of IP addressing and subnet division |
| **DHCP & DNS Server** | Configuration of DHCP and DNS services |

> New projects will be added over time.
